- **品种**: 黄金(AU)、白银(AG)期货

## 策略说明

### 1. Dual Thrust 突破策略 (趋势行情)

**核心逻辑**: 利用开盘范围 + 历史高低点构建通道，突破入场

**参数**:
- K1/K2: 上/下通道系数 (默认 0.4/0.6)
- N: 回溯周期 (默认 4)
- 周期: 5分钟

**信号**:
- 价格 > Open + K1 * Range → 做多
- 价格 < Open - K2 * Range → 做空

### 2. 均值回归策略 (震荡行情)

**核心逻辑**: 布林带 + RSI 超买超卖

**参数**:
- 布林带: 20期, 2倍标准差
- RSI: 14期
- 超买: 70, 超卖: 30

**信号**:
- 价格触及下轨 + RSI < 30 → 做多
- 价格触及上轨 + RSI > 70 → 做空

### 3. 组合策略 (自适应)

**逻辑**: 根据 ADX 判断市场状态，自动切换策略
- ADX > 25 → 使用突破策略
- ADX < 20 → 使用均值回归策略

## 风控规则

| 参数 | 值 | 说明 |
|------|-----|------|
| 单笔风险 | 1-2% | 总资金百分比 |
| 最大持仓 | 20% | 单品种最大仓位 |
| ATR止损 | 2-3倍 | 动态追踪止损 |
| 日内平仓 | 必须 | 不留隔夜仓 |
| 最大亏损 | 5% | 日最大亏损，触发停止交易 |

## 时间安排

|时段 | 时间 | 策略倾向 |
|------|------|---------|
| 日盘 | 09:00-11:30 | 均值回归 |
| 下午 | 13:30-15:00 | 观察/轻仓 |
| 夜盘 | 21:00-02:30 | 突破跟踪(活跃时段) |
